# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play the classic Tic Tac Toe game in the browser.

## Features

* Two-player gameplay (X and O)
* Winner detection
* Draw detection
* Interactive user interface
* Prevents overwriting occupied cells
* Responsive and easy-to-use design

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

* `index.html` – Game layout and structure
* `style.css` – Styling and design
* `script.js` – Game logic and winner detection
